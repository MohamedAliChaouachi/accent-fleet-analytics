\chapter{Python Scripts and Figure Generation}
\label{app:python}

This appendix references the Python scripts used in the project. The full versions are stored under \texttt{scripts/python/}.

\section{Automated extraction to CSV}
\label{sec:app-py-extract}
The script \texttt{extract\_to\_csv.py} reads a manifest of SQL queries from \texttt{config/queries.yaml}, executes each query through a SQLAlchemy engine and writes the result as a timestamped CSV in \texttt{exports/}. The script accepts a \texttt{--dry-run} flag for unit-testing purposes.

\section{Figure generation}
The figures of the thesis are produced by self-contained Python scripts located in \texttt{scripts/python/figures/}. Each script reads its input either from the database (read-only credentials) or from a CSV produced by \texttt{extract\_to\_csv.py}, and writes a publication-quality PDF in \texttt{report/figures/}. \section{Modeling scripts}
The modeling notebooks are stored under \texttt{notebooks/04\_modeling/}. The notebook \texttt{01\_device\_behavior\_clustering.ipynb} fits the per-tenant K-Means models and exports the artefacts; the notebook \texttt{02\_anomaly\_risk\_score.ipynb} fits the per-tenant Isolation Forest models and computes the rescaled risk scores.
